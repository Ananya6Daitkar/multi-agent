# FutureBoard Submission

## Basic Information

Project Title: FutureBoard: Enterprise Decision Immune System

Short Description: FutureBoard is a Band-powered multi-agent decision room where specialized agents simulate, red-team, negotiate, and remember high-stakes enterprise decisions before leaders act.

Technology Tags: Band, Multi-Agent Systems, Agent Coordination, Enterprise Workflows, Decision Intelligence, Risk Simulation, Featherless AI, AI/ML API, Groq, High-Stakes Workflows.

## What To Demo

1. Open the app.
2. Click `Run Band Immune Response`.
3. Show the live Band chat room transcript.
4. Show handoffs, disagreement, escalation, negotiated plan, and memory.
5. Explain that external Band Agent API is attempted through official `/api/v1/agent` endpoints and local fallback keeps the demo reliable.

## Deployment

Start command:

```bash
npm start
```

Required environment variables:

```env
BAND_API_KEY=
BAND_BASE_URL=https://app.band.ai
groq_API_KEY=
FEATHERLESS_API_KEY=
FEATHERLESS_BASE_URL=
FEATHERLESS_MODEL=
```
